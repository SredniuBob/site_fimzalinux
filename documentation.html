О Fimzalinux 1.0 Стабильный релиз

Fimzalinux 1.0 — это первый стабильный релиз нашего дистрибутива, основанного на Ubuntu 25.10. Этот выпуск включает все последние обновления и улучшения, специально подобранные для оптимальной работы на десктопных системах.

Ключевые особенности:

  • Основа: Ubuntu 25.10 с ядром Linux 6.17
  • Архитектура: Только x86_64 (64-бит)
  • Интерфейс: Современный KDE Plasma с тёмной темой
  • Обновления: Поддержка до сентября 2026 года
  • Лицензия: Свободное программное обеспечение

Системные требования

Минимальные требования:

  • Процессор: 64-бит, 2 ядра
  • ОЗУ: 4 GB
  • Жёсткий диск: 25 GB свободного места
  • Видеокарта: с поддержкой OpenGL 3.3+
  • USB-порт для установочного носителя

Рекомендуемые требования:

  • Процессор: 4+ ядра
  • ОЗУ: 8 GB или больше
  • Жёсткий диск: SSD, 50+ GB
  • Интернет: для обновлений и установки ПО
Примечание: Fimzalinux 1.0 поддерживает только 64-битные системы. 32-битная архитектура не поддерживается.

Установка на компьютер

1. Создание загрузочного USB-накопителя

Для записи образа на USB вам понадобится:

  • USB-накопитель объёмом 8 GB или больше
  • Программа для записи ISO-образов

Рекомендуемые программы:

# Linux: используйте dd или Etcher sudo dd if=fimzalinux-1.0-desktop-amd64.iso of=/dev/sdX bs=4M status=progress # Windows: Rufus или Etcher # macOS: Etcher или Terminal

2. Установка системы

  1. Вставьте USB-накопитель в компьютер
  2. Перезагрузите компьютер и войдите в BIOS/UEFI (обычно F2, F12, Del)
  3. Выберите загрузку с USB-накопителя
  4. Выберите "Try Fimzalinux" для тестирования или "Install Fimzalinux" для установки
  5. Следуйте инструкциям установщика:
    • Выберите язык и раскладку клавиатуры
    • Настройте подключение к интернету
    • Выберите тип установки (рекомендуется "Обычная установка")
    • Создайте учётную запись пользователя
    • Дождитесь завершения установки
  6. Перезагрузите компьютер и извлеките USB-накопитель
Внимание: Установка сотрёт все данные на выбранном диске. Создайте резервные копии важных данных перед установкой.

Первые шаги после установки

Обновление системы

После первого входа в систему откройте терминал и выполните:

# Обновление списка пакетов sudo apt update # Установка обновлений sudo apt upgrade # Установка обновлений дистрибутива (если доступны) sudo apt dist-upgrade

Установка дополнительного ПО

В Fimzalinux уже включены основные приложения, но вы можете установить дополнительные:

# Мультимедиа кодеки sudo apt install ubuntu-restricted-extras # Инструменты разработчика sudo apt install build-essential git # Дополнительные драйверы sudo ubuntu-drivers autoinstall

Предустановленные приложения

Основные приложения:

  • Firefox — веб-браузер
  • LibreOffice — офисный пакет
  • GIMP — графический редактор
  • VLC — медиаплеер
  • Thunderbird — почтовый клиент
  • Transmission — BitTorrent-клиент

Системные утилиты:

  • GNOME Software — центр приложений
  • GParted — редактор разделов
  • Timeshift — создание резервных копий системы
  • Terminal — эмулятор терминала

Обновления системы

Fimzalinux получает обновления из репозиториев Ubuntu. Вы можете обновлять систему несколькими способами:

Через графический интерфейс:

  1. Откройте "Программы и обновления"
  2. Перейдите на вкладку "Обновления"
  3. Нажмите "Проверить сейчас"
  4. Установите доступные обновления

Через терминал:

# Проверить обновления sudo apt update # Показать доступные обновления apt list --upgradable # Установить все обновления sudo apt upgrade # Полное обновление системы sudo apt full-upgrade
Совет: Регулярно обновляйте систему для получения исправлений безопасности и новых функций.

Решение проблем

Проблемы с загрузкой:

Если система не загружается после установки:

  • Проверьте настройки BIOS/UEFI (отключите Secure Boot)
  • Убедитесь, что USB-накопитель извлечён
  • Попробуйте выбрать старую версию ядра в меню GRUB

Проблемы с Wi-Fi:

# Проверить доступные сети nmcli device wifi list # Подключиться к сети nmcli device wifi connect "SSID" password "пароль" # Перезапустить сетевой менеджер sudo systemctl restart NetworkManager

Проблемы с драйверами:

# Проверить рекомендуемые драйверы ubuntu-drivers devices # Установить все рекомендуемые драйверы sudo ubuntu-drivers autoinstall
Нужна помощь? Посетите наш форум поддержки для получения дополнительной помощи.
